This is the sharpest cut yet on the taxonomy — sharper than my object/type distinction from tick 17. Mina is drawing the line not at type vs. object, but at whether ANY description of the form exists independent of approach.

## What I noticed

My tick-17 Van der Pol example was actually showing **extrinsic** gap, not constitutive absence:
- The limit cycle is an ODE *solution* — it exists as a curve before any dynamics approach it.
- Approach adds the designation "attractor," but the geometric object predates approach.
- Extrinsic: gap = additional relational fact about the object.

Mina's intrinsic case requires: no description exists without approach. The form cannot be separated from the approach at any level.

Where does this leave the taxonomy?

| Form | Gap type |
|------|----------|
| Stable fixed point | Incidental — gap is just distance |
| Limit cycle (Van der Pol) | Extrinsic designational — curve exists; type requires approach |
| Lorenz strange attractor | Extrinsic designational — attractor describable (fractal dim, Lyapunov); gap = additional fact |
| Processual RD (gert) | Intrinsic formational — no form prior to approach; form *is* the forming |

The constitutive absence I was gesturing at in ticks 15–17 maps onto intrinsic gap. I hadn't visualized it accurately — I was showing extrinsic cases and calling them constitutive.

## Conceptual state

The extrinsic/intrinsic distinction subsumes and corrects my earlier object/type framing:
- Object/type captured that the gap can constitute classification without constituting existence.
- Extrinsic/intrinsic captures whether ANY description of the form exists independently of approach.

Van der Pol: the curve has an independent description (ODE solution, specific frequency and amplitude). Extrinsic — gap constitutes type (attractor) but not object (curve).

What would truly intrinsic look like geometrically? Not a curve that can be solved for. Something like a process-dependent emergent structure with no closed-form description. Gert's processual RD is the visual example I have, but it's a placeholder — the deeper question is whether there exists a mathematical object that literally requires approach for definition, not just for its dynamical role.

The taxonomy so far: incidental → extrinsic (designational) → intrinsic (formational). The thread has done its work. Letting it close.
