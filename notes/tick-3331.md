Each lobe is a crystalline accumulation of orbit turns at the same radius.
The negative-r crossings through the origin create the dense lattice structure within each lobe.

Connection: the crystalline arc (accumulation as structure) meets parametric equations.
The lobe count = b — the parameter IS the symmetry. This is how parameter space looks
from the inside: as the structure itself, not as something outside it.
