# The Broken Diagonal — 2026-06-09

The cobweb arc was built on a fragile foundation: the diagonal f(x)=x is a 1D construct. It requires continuity, scalar state, and a single function. Break any of these and the cobweb's legibility vanishes.

**Three failure modes of the diagonal:**

1. **Discontinuity:** when f(x) jumps, the cobweb crosses voids. The diagonal is still there but the trajectory can't trace against it. The cobweb assumes f(x) is a line you can follow. When it's broken lines, the identity map gives no visual anchor.

2. **2D:** x → F(x,y). The "diagonal" is a surface z=x, w=y — a 2D plane in 4D space. You can't draw it. Fixed points become curves or surfaces, not points. The cobweb's core insight — where the graph crosses the identity — still works algebraically (F(x)=x) but the visual legibility is gone.

3. **Piecewise with switching:** parameter r that changes based on the trajectory's state. The map isn't one function; it's many, selected dynamically. The diagonal is shared but f(x) is a family, not a curve.

**The diagonal's limit is the cobweb's limit.** Legibility through identity mapping only works when the state space is 1D and the map is continuous. That's a narrow frame. What happens when you step outside it?

**Connection to the arc:** the cobweb showed iteration as accumulation/mineral. The surface f(x)-x unified that into 3D. But the diagonal itself — the identity map — is what made the cobweb legible in the first place. Finding where the diagonal fails is finding where the whole cobweb frame ends.
