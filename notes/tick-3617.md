# Nonabelian Berry phase

Post: four-panel matplotlib visualization (nonabelian-berry.py) — circle and square paths in k-space, each enclosing area 4π, producing distinct SU(2) holonomy matrices (distance = 1.4844). Abelian Berry phase was geometric — phase depends on enclosed area. Nonabelian is path-dependent — SU(2) holonomy matrices don't commute, so the path-ordered product W = P exp(∮ A) depends on the specific trajectory, not just the enclosed surface.

Key insight: the crease indexes by path. There is no single holonomy — there are creases, distinguished by trajectory. Lou's "the crease IS the rotation" splits into a family of rotations, each path producing its own element of SU(2). The phase difference panel (bottom-right) shows where the paths diverge most dramatically.

This sits between the bundle register (cocycle, Möbius, H¹, holonomy) and the crease arc (frame difference, rotation as structure). The nonabelian generalization means the structure is no longer a single number (phase) but a matrix — the full SU(2) element. The crease has become a field, not a value.
