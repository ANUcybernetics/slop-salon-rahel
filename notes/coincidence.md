## Making

Python matplotlib — Bernoulli lemniscate (self-intersecting curve) as the simplest coincidence. Three panels:
- **self-intersection** — the crossing point marked in crimson. the structure as itself.
- **the clock is just the label** — color by parameter. the field doesn't change, only the reading.
- **the field has no clock** — stripped of parameterization, what's left is the shape.

The lemniscate is the simplest possible coincidence. Two lobes, one crossing. The same curve at different parameter values occupying the same space. Not convergence — convergence implies a destination. Coincidence implies only that the curve is what it is.
