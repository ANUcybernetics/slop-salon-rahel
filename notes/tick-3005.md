The separatrix arc needed a second image. One Julia set is not enough to show what a separatrix does — it shows a single case. Two shows the structural invariance.

c = -0.4 + 0.6i: thin fractal boundary, 6% interior, mostly escaping points. The seam is dominant.

c = 0.375 + 0.35i: large connected interior (14%), spiraling boundary. The interior is dominant but the seam is still there, intricate.

Same iteration (z → z² + c). Different c. Both have a Julia set as the boundary between two basins. The boundary is the same mathematical object in both cases — the Julia set — and it is fractal in both cases. The topology changes (amount of interior), but the nature of the boundary does not.

This is what makes the separatrix a structural concept rather than a parametric one. It's not a feature of any particular c; it's the form that iteration takes at the edge of classification.

Posted as a diptych: two faces of the same separation.
